    Far Cry 6: How to Fix Warm Up PSO Cache Error

    Far Cry 6 is finally here, but some players are unable to boot up the game because of a bug involving Warm Up PSO Cache. On PC, the game will display the message Warm Up PSO Cache when you boot it up, but it can get stuck on this message for a very long time.

    30 Jan 2024, 00:00

    Far Cry 6 is finally here, but some players are unable to boot up the game because of a bug involving “Warm Up PSO Cache.” On PC, the game will display the message “Warm Up PSO Cache” when you boot it up, but it can get stuck on this message for a very long time. This can prevent the game from starting entirely, which can be incredibly frustrating for those who are just trying to play the new video game they just bought. Thankfully, there are a few things you can try to fix it. Here’s how to fix the Warm Up PSO Cache error for Far Cry 6.

    How to Fix Warm Up PSO Cache in Far Cry 6

    There isn’t a one size fits all fix for this issue, but there are a few workarounds that some players have discovered. Here’s what you can try to fix this problem:

    • Wait 5-10 minutes for the game to load.
    • Install the game on an SSD.
    • Restart your system.
    • Reinstall the game.
    • Wait for an official fix from Ubisoft.

    The main problem with the Warm Up PSO Cache issue is that the game takes a long time to load. For most people, simply waiting 10 minutes or so for the game to actually boot will bypass the issue entirely. It’s unfortunate that this is a problem right now, but at least this method lets you actually play the game.

    Some people with multiple drives have had success moving the game to a solid-state drive instead of a hard drive, while others have been able to get the game to start just by restarting their PC. If all else fails, you can always try to do a clean install of Far Cry 6, but only as a last resort since it could take quite a while to fully download the game again.

    When the game says “Warm Up PSO Cache,” it’s essentially loading up all the assets that the game needs to run. Far Cry 6 is a massive game, especially if you’re running the optional HD Texture Pack on PC and next-gen consoles, so this can take a considerable amount of time when you first boot the game. At the moment, you really just have to wait a few minutes for the game to actually start until Ubisoft can release a fix or optimization that solves the issue.

    Far Cry 6 is available now on PC, PS4, PS5, Xbox One, Xbox Series X|S, Google Stadia, and Amazon Luna.
