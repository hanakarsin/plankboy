    14 Feb 2024, 00:00

    Minecraft Update 1.9 will be released as a final version next week on Monday, February 29th. However, those who want to try out the long-in-development patch can do so by grabbing the pre-release version, which is available now. Minecraft Version 1.9 has been the longest stretch between new versions of the game, so many are expecting some big changes for this update.

    The patch notes haven’t been revealed yet, though they are promised when the update launches next week. However, Mojang did give some idea of what has been altered by this patch. “Bugs fixed. Increased durability of shields. Some crystals in the end are protected by cages again. Added 34.7% more hype for Minecraft 1.9.” That last one seems to be the most important, as Mojang is pushing Minecraft 1.9 as a big shift for the game.

    This update will, for a time, only be available on PC, with console versions of Minecraft lagging behind a bit due to various reasons. However, this is still big news for console fans of the game, as it means they’ll be getting this update soon as well. It’ll have to be shifted to the consoles first, and then go through the various certification procedures that Microsoft and Sony require, but it’s still worth getting hyped over.
