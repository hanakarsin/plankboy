    Resident Evil 7 PC System Requirements Revealed

    While were still several months away from the release of Resident Evil 7 Biohazard, we already have the official PC system requirements for the game. The details were uploaded to the games Steam page, which provides both the minimum and recommended requirements.

    16 Jun 2024, 00:00

    While we’re still several months away from the release of Resident Evil 7 Biohazard, we already have the official PC system requirements for the game. The details were uploaded to the game’s Steam page, which provides both the minimum and recommended requirements.

    The following are both the minimum and recommended PC system requirements for Resident Evil 7 Biohazard:

    MINIMUM:

    • OS: WINDOWS 7,8,8.1,10 64-BIT
    • Processor: Intel Core i5-4460, 2.70GHz or AMD FX-6300 or better
    • Memory: 8 GB RAM
    • Graphics: NVIDIA GeForce GTX 760 or AMD Radeon R7 260x
    • DirectX: Version 11
    • Additional Notes: 1080P CPU/GPU at 30FPS Players will have to either lower texture or set streaming textures to OFF in order to optimize performance.

    RECOMMENDED:

    • OS: WINDOWS 7,8,8.1,10 64-BIT
    • Processor: Intel Core i7 3770 3.4GHz or AMD or better
    • Memory: 8 GB RAM
    • Graphics: NVIDIA GeForce GTX 960 or AMD Radeon R9 280X or better
    • DirectX: Version 11
    • Additional Notes: 1080P CPU/GPU at 60FPS

    The game most recently got what was called the “Twilight” update to the Beginning Hour demo that was released after its E3 2016 reveal, which added new things to the demo for players to discover. You can check out everything that has been discovered so far in it by clicking right here.

    Resident Evil 7 Biohazard is set to launch for PlayStation 4, Xbox One and PC on January 24th, 2017.
