    To render hundreds of soldiers in a civil and good-looking manner on your screen it seems evident that you need quite some horsepower in your gaming rig. If you’re having doubts whether or not your system will be able to run Total War Rome 2 even at lower settings you can take a look at the system requirements shown below.

    In case your machine doesn’t have the chops, there’s still some time left to upgrade as the game is set to be released on the 3rd of September this year.

    Minimum specifications:
    OS: XP/ Vista / Windows 7 / Windows 8
    CPU:2 GHz Intel Dual Core processor / 2.6 GHz Intel Single Core processor
    Memory: 2GB RAM
    GFX:512 MB DirectX 9.0c compatibel
    DirectX®:9.0c
    HDD: 35 GB HD ruimte
    Resolution: 1024×768

    Recommended specifications:
    OS: Windows 7 / Windows 8
    CPU:2nd Generation Intel Core i5 processor (or more)
    Memory: 4GB RAM
    GFX:1024 MB DirectX 11 compatibel
