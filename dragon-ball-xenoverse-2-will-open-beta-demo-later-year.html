    If you cannot wait for Dragon Ball Xenoverse 2, you might be happy to know that it might be playable very soon. You could get a chance to play it before it comes out this October.

    As noted by Saiyan Island, a Bandai Namco Japan representative has announced that there will be an open beta for Dragon Ball Xenoverse 2. This means you can play a demo to test out the network capabilities of the game.

    If you remember correctly, the first Dragon Ball Xenoverse video game also had a beta prior to its release. Unfortunately, the Bandai Namco Japan representative didn’t specify when the sequel’s beta will be available, but it should be out a few weeks before it comes out in late October.

    Bandai Namco probably doesn’t want to repeat the mistakes it made with the first game. The servers on launch day for the first game were atrocious. The servers were down for several hours in the first week it came out.

    Doing an open beta for the second game is a good idea to help improve the network environment. There have been many botched game launches this generation so game developers have been extra careful recently not to repeat the same mistakes.
