    Call of Duty: Infinite Warfare is a big game, in more ways than just its inevitable sales. Rumors had been putting the full experience of Infinite Warfare and Modern Warfare Remastered at 130GB, and according to the newly release file size on PC that seems to be completely accurate. The Call of Duty: Infinite Warfare PC system requirements, including the file size required for install, are below.

    Call of Duty: Infinite Warfare PC System Requirements

    OS: Windows 7 64-Bit or later
    Processor: Intel Core i3-3225 @ 3.30GHz or equivalent
    Memory: 8 GB RAM
    Graphics: NVIDIA GeForce GTX 660 2GB / AMD Radeon HD 7850 2GB
    DirectX: Version 11
    Network: Broadband Internet connection
    Storage: 70 GB available space
    Sound Card: DirectX 11 Compatible

    These are just the minimum requirements too, with no official recommended specs released just yet. That means that a machine with these capabilities will simply be able to run the game, not that it will actually run well. 1080p and/or 60 fps will definitely require more power than this, so hopefully you have your upgrades installed already and are good to go. I mean, there have been a lot of big PC releases already this month, so hopefully you do.

    Call of Duty: Infinite Warfare hits PS4, Xbox One, and PC on November 4th.
