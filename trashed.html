    How to Trigger All Heroic Public Events in Destiny 2

    In Destiny 2 there are new modifiers to the Public Event system with the ability to trigger a Heroic version of the event upon completing certain objectives.  For every different Public Event in the game there is a modifier of it if you can complete these tasks.  The challenges that players will face are much harder with increased enemy counts and high-level enemies.  With the greater challenge come greater rewards upon completion.  While some of these Heroic Events can be performed with little teamwork, it is recommended that you try to tackle these with a fireteam as some of them require cooperation between multiple players.

    Cabal Excavation Heroic Public Event

    During the Cabal Excavation Event you’ll be tasked with capturing a drill in the standard version, fending off wave after wave of Cabal enemy.  To trigger the Heroic Public Event you need to shoot down the small Cabal ship called a Thresher that will make numerous passes during the event.  Once you shoot down the ship, the drilling machine will shut down and you’ll no longer need to stay within the circle to complete the event.  Instead, a powerful Cabal (Excavation Valus) leader will drop onto the battlefield and you must defeat them within the allotted time to beat the Heroic Event.

    Glimmer Extraction Heroic Public Event

    During the Glimmer Extraction Public Event the normal mode will feature four waves of extraction.  I you destroy the small extraction machines that surround the site you’ll trigger the Heroic Event which has you looking to transmat the Glimmer for yourselves.  There are numerous structures in this area, so look for the ones that you can literally see the blue Glimmer mats being sucked into, (they’ll also show damage numbers).  You’ll need to hold off multiple waves of enemies at this point, with an Epic Fallen Captain with a powerful stomping ability entering the battlefield making it harder and harder to stay in the circle and transfer the Glimmer to yourself.

    Ether Resupply Heroic Public Event

    In Firebase Hades in the EDZ you’ll find the Ether Resupply Public Event and probably the easiest of the Heroic Events to trigger in Destiny 2.  Here, you’ll find Servitors that are stealing Ether for the Fallen army. Simply focusing down all the Servitors will trigger the Heroic Event which is simply a harder version of the normal event.  The only trick here is that you need to look out for Servitors hiding in the base as sometimes they can be obscured by terrain in the base.  You’ll have a limited amount of time to take out each of the Servitors that spawn into the area.

    Injection Rig Heroic Public Event

    The Injection Rig is one of the trickiest Heroic Public Events in Destiny 2.  This is the one that features a massive Cabal structure that forces you out of an enclosed area or risk the chance of dying.  This is what makes the event so tricky.  The only way to trigger the Heroic Public Event in Injection Rig is to stay inside the dome while the heating vents of the large machine are open.  Using teamwork is really the only way to get this event done. Once you kill a Psion you’ll only have a small window of opportunity to hit the vents at the top of the machine while they’re open and your health won’t allow you to withstand the heat very long.  Once you do destroy the vent at the top of the machine, you’ll need to repeat this tasks for the bottom vents and this will trigger the Heroic Event.

    Arsenal Walker Heroic Public Event

    At first glance the Arsenal Walker Public Event seems like the same old thing from Destiny 1, but to unlock the Heroic Version is a little harder than just defeating the Walker.  As you’ll notice while fighting the giant spider tank, when you shoot off the armor on its legs like normal, orbs will drop to the ground.  There are multiple weapon caches around the Walker that can be triggered to unlock a powerful cannon.  To get inside these shielded caches you’ll need to take the orbs to a device next to cache and insert it.  Do that for all of them and you’ll trigger the Heroic Version of this event.  Once triggered a second Walker will be dropped into the battle, you’ll have to deal with the a Cabal Major named Thox.

    Vex Construction Heroic Public Event

    For those familiar with the beginning of the Vault of Glass Raid, the Vex Construction public event will be familiar territory.  The normal version of this event requires that you kill the Vex before they can sacrifice themselves on three plates around the Construct.  Standing on these plates will allow you to build a meter that captures it.  Triggering the Heroic version of this event requires that you capture each of the three plates.  You can expect a massive Hydra during the Heroic version of this event, as well as many more Vex to deal with.

    Witches Ritual Heroic Public Event

    The Witches Ritual on Titan offers another chance at a Heroic Event.  To trigger this one you’ll need to take out two shielded crystals that sitting above the portal.   To bring down the shields you need to stand in the Hex circles and can then destroy them.  Doing so will start the Heroic version of this event in which will summon a Knight known as the Abyssal Champion.  At this point, it’ll become a race against the clock to take down this powerful enemy and complete the Heroic Event.
