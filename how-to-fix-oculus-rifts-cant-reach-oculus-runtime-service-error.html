    In case you weren’t already aware, every Oculus Rift in the world was broken yesterday due to an expired certificate that prevented running the software required for the headset. There’s a fix now, but the same error means it can’t be automatically pushed to everyone like it usually would. Oculus has provided the steps though, so here’s how to fix Oculus Rift’s “Can’t Reach Oculus Runtime Service” error.

    Go to Oculus’s official patch site here. Click Download Install Patch to get the new update. Run the OculusPatchMarch.exe file that you receive. If you receive any prompts or warnings simply click Yes to get through them, the file is safe. When the file runs, click Repair to run it and fix your system. Launch the Oculus app and you should soon be prompted to update. Let it do so and you should be back in your Oculus Rift VR world soon.

    This was a very unfortunate scenario for one of VR’s most important companies. Many have wondered how such an error could occur, but no one has given a solid answer just yet. It is clear that the security features the Oculus uses were one of the culprits, so this has stoked the fires of Oculus opponents quite a bit.
