    All Command Prompts for BedWards in Roblox

    If you’re looking to have a full list of all of these different commands available at all times, make sure that you’re keeping this page bookmarked. Here is every command currently available in BedWars for Roblox!

    Custom Match Commands List

    • /setac enabled – enable anti-cheat
    • /setac disabled –disable anti-cheat
    • /void – teleport to the Void
    • /bedwars – return to the BedWars lobby
    • /Islands – go to Roblox Islands
    • /setTeam (player | @all | @team:(team_name)) (team_name) – set the team of individual player or players.
    • /setHealth (player | @all | @team:(color) | @s) (health) – set the health of a player, players, or the whole team.
    • /disaster (“meteors” | “void_rise” | “toxic_rain” | “tornado”) – trigger natural disasters.
    • /give (player) (@all) (@team: team_color) (item) (item amount) – give the player, all players, or a certain team an item to use.
    • /sethealthregen (player) (regen_amount) – set the health regen for a specific player.
    • /cohost all – grant cohost permissions to everyone.
    • /enchant (enchantmentname_level – e.g., /enchant static_3) – grant an enchantment of your choosing to a player.
    • /tp (player) – Teleport to a specific player on the map.
      • /tp (target_players) (destination_player) – teleport different players to a certain player.
    • /tpall – Teleports everyone to the user
    • /tpbed (Ex. /tpbed red) – Teleports you to the specific bed team color you choose when entering the command.
    • /spawn (item name) (amount) – Spawn items, with the amount that you choose 
    • /announce (message) – Broadcasts a message in a popup window to all players
    • /kick (player) – Remove players from your game, kicking them from it. You will need to use their Roblox Username.
    • /setgeneratormultiplier (amount) – Will change the speed of the Iron, Diamond, and Emerald generator. 
    • /blacklistItem (item name) – Will not let players purchase this item from the Item Shop, will need to start a new game to reenable the item. 
    • /disablekit (kit name) – Will not allow players to use a specific kit during the game. Will need to start a new custom game to enable this kit. 
    • /setdeathmatchtime (seconds) – How long the game will continue before sudden death begins. 
    • /togglespawn (item name) (true/false) – Will let you select if an item will spawn, true for yes, false for no.
    • /luckyairdrop – spawn a Lucky Airdrop at a random point on the map. 

    Chat Commands List

    • /hub – Will bring you back to the BedWars server room, allowing you to exit the match you are in and find a new area to start in.
