    For Honor Update 2.31.0 Patch Notes

    Update 2.31.0 has arrived for For Honor, and heres the full list of changes and fixes added with this patch. Ubisoft has now deployed a new update for For Honor that is rolling out on all platforms. Maintenance for this patch is still going and it will last for nearly two hours.

    03 Jun 2024, 00:00

    Update 2.31.0 has arrived for For Honor, and here’s the full list of changes and fixes added with this patch.

    Ubisoft has now deployed a new update for For Honor that is rolling out on all platforms. Maintenance for this patch is still going and it will last for nearly two hours.

    The file size is 2.65 GB on PC, 1.78 GB on PS4 and also 1.78 on Xbox. The update itself has many gameplay improvements, balancing as well as the usual bug fixes. You can check out the full details of this patch posted down below.

    For Honor Update 2.31.0 Patch Notes

    BUG FIXES
    FIGHTERS

    Gladiator

    • [Bug Fix] Fixed an issue that caused the victim of skewer to have a guaranteed Guard Break on the Gladiator after getting hit by Skewer near a Wall, increased the Changed ”Exit On Wall” for the victim to 500ms (from 400ms) (FH-2320)

    Shaolin

    • [Bug Fix] Fixed an issue that caused entering “Qi Stance” from a Hit or Blocked right neutral heavy while holding right or forward to results in Left Stance Animation (FH-2323)
    • [Bug Fix] Fixed an issue that caused the Shaolin to have a wrong animation when moving guard direction during a parry (FH-2325)

    Jiang Jun

    • [Bug Fix] Fixed an issue that caused the Jiang Jun hard feints to snap to opponents (FH-1893)
      Hitokiri
    • [Bug Fix] Fixed an issue that caused the Hitokiri’s side dodge to have an incorrect animation at the end of the dodge (FH-2147)

    Zhanhu

    • [Bug Fix] Fixed an issue that caused the Zhanhu’s right Unblockable Finisher (from left neutral heavy) to have an incorrect transition frame when feinted to neutral (FH-2099)
    • [Bug Fix] Fixed an issue that caused the Zhanhu to be able to cancels the Finisher Attacks with a Dodge or Dodge Attack

    Kyoshin

    • [Bug Fix] Fixed an issue that caused the Kyoshin “Kaze Stance” branching from Sprint Attack to finishers, (and from Sprint Attack to Kaze Stance), to be longer than expected, reduced to 200ms recovery (from 266ms) (FH-2310)
    • [Bug Fix] Fixed an issue that caused the Kyoshin “Kaze Stance” and “Aoarashi” to flicker (FH-2243)
    • [Bug Fix] Fixed an issue that caused the Kyoshin “Hakaze” Superior Block to connect to a Nobushi after specific moves (FH-2243)
    • [Bug Fix] Fixed animation issue that caused the Kyoshin “Fujin Cuts” to play an incorrect animation if stopped after the first attack (FH-2302)
    • [Bug Fix] Fixed an issue that caused the Kyoshin “Fujin Force” to not track Soldiers (FH-2259)

    FEATS

    • [Bug Fix] Fixed an issue that caused the Kyoshin to heal when hitting the Ram when the “Hikarimore” Feat or “Kirari To Kage” Feat were active (FH-2292)

    MAP
    Beachhead

    • [Bug Fix] Slightly adjusted spawn positions for Defenders and Attackers

    Sanctuary Bridge

    • [Bug Fix] Fixed an issue that could cause Players to get stuck near the trap lever at Capture Zone A (FH-2280)

    APPRENTICE TRIAL: STAMINA

    • [Bug Fix] Adjusted the first Warlord Bot’s behavior to be easier to defeat using the intended method and added a text explaining to the player what to do
    • [Bug Fix] Lowered difficulty of the 2nd Warlord Bot and final encounter

    SOUND

    • [Bug Fix] Fixed an issue that caused the Executions sound effects and voice lines to be missing in Dominion Beachhead with headphones on Xbox (FH-1880)
    • [Bug Fix] Fixed an issue that caused the “Incalescent Memory” effect audio to not loop while the character is idling (FH-2287)

    CUSTOMIZATION

    • [Bug Fix] Fixed an issue that caused the Zhanhu’s “Horkos Qipian” chest to have incorrect physics on the default body type (FH-2271)
    • [Bug Fix] Fixed an issue that caused the Hitokiri’s walking animation to behave incorrectly in some situations (FH-1945)
    • [Bug Fix] Fixed an issue that caused the Nobushi’s “III Winds” ornament to float or clip on some helms (FH-1977)
    • [Bug Fix] Fixed an issue that caused the Orochi’s standards on some of their chest armour to not match the cloth around their hips (FH-1572)

    The following information you see above was originally posted on the official Ubisoft website. You can click the link provided for more into about the patch. For Honor is out now for PC, PS4 and Xbox One.
